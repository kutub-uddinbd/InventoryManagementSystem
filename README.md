# InventoryManagementSystem
This an Inventory Management System Application.
This Application has 3 menu,
  >Product list,
  >Purchase product,
  >Sale Product.
we can easily add Products, Purchase Products,and sale Products.
This is a Simple application
Use Technology:
  >MS SQL Server 2016,
  >Visual Studio 2019,
  >MVC 5,
  >Database First.
